Maze Runner¶

maze_runner.make_maze(w: int = 200, h: int = 200, start: bool = False, end: bool = False) → str[source]¶

Creates a maze of the given width and height, with options to randomly set start and end locations for the exploration procedure.

Shamelesly taken from http://stackoverflow.com/a/31634316/1245214

Parameters:
  • w – width of the maze
  • h – height of the maze
  • start – whether to set a start position in the maze for the search algorithms
  • end – whether to set an end position in the maze for the search algorithms
Returns:

a maze in its string representation
maze_runner.solve(maze: str, strategy: str = 'dfs', verbosity: int = 0, file: str = None)[source]¶

Solves a maze using one of the available algorithms.

Parameters:
  • maze – a maze in its string representation
  • strategy – which algorithm to use. Either ‘dfs’, ‘bfs’ or ‘a*’
  • verbosity – verbosity level of the exploration procedure. From 0 to 3.
  • file – Path to save an image file of the exploration procedure.
